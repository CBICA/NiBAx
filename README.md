# iSTAGING-Tools

## Setup

### Linux (CUBIC)
```shell
python -m venv .env
.env/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt 
```

### Windows 10
```shell
python -m venv .env
& .env/Scripts/Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt 
```
